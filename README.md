# calgos

Comparison to Koura's algorithms implemented in zig.. now in C.

## Getting started

Install a C compiler.

tl;dr

```
make love
bin/quicksort
# ??
echo profit
```

Create the build system:

```
make generate
```

Build the project:

```
make build
```

Install:

```
make install
```

Run a target:

```
bin/quicksort
```
